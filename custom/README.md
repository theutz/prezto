# Custom Modules

Custom modules to load alongside the default prezto modules. Make sure to
append `$ZPREZTODIR/custom` to `zstyle ':prezto:load' pmodule-dirs` in
`.zpreztorc`.

## Chezmoi

Integrates [chezmoi](https://chezmoi.io) into zsh.

## FZF

Terminal-based generic fuzzy finder.

## LazyGit

Tools for integrating lazygit into `zsh`

## LF

Integate the lf file manager into zsh.

## Keychain

Integrates Gentoo's keychain package into zsh.


## Zoxide

Setup and configure zoxide
